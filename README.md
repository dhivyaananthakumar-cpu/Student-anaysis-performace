 Student Performance Analysis

 Overview
This project analyzes the academic performance of 250 students using data analysis and machine learning techniques. It helps identify trends, top performers, and predict student outcomes.


 Dataset
- Total Students: 250
- Features:
  - Student ID
  - Department
  - Subject Marks
  - Total Marks
  - Average
  - Grade



Objectives
- Analyze student performance
- Identify top and low performers
- Compare department-wise results
- Predict student performance using Machine Learning


 Project Highlights
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of performance trends
- Machine Learning model for prediction



 Data Preprocessing
- Handled missing values
- Converted data types
- Calculated Total and Average marks
- Assigned grades



 Exploratory Data Analysis (EDA)
- Distribution of marks
- Department-wise performance
- Top 10 students analysis
- Grade distribution



 Machine Learning Model
- Models Used:
  - Linear Regression
  - Decision Tree
- Input: Subject marks
- Output: Total marks / Grade prediction



Visualization
- Bar Charts (Department averages)
- Pie Charts (Grade distribution)
- Line Graphs (Performance trends)



 Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn



Results
- Identified top-performing students
- Found weak subject areas
- Achieved good prediction accuracy


 Pipeline
1. Load dataset
2. Preprocess data
3. Perform EDA
4. Train ML model
5. Evaluate results



 Generating Predictions
- Input new student marks
- Model predicts total marks or grade
- Useful for performance evaluation
